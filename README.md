NestJS 練習用リポジトリ

[公式｜NestJS](https://nestjs.com/)

## 特徴
- Node.js（+ TypeScript） のフレームワーク
- Angular ライクなフルスタックでスケーラブルなバックエンドフレームワーク
- MVC ベース
- REST API, GraphQL API などに活用

## 環境構築

NestJS CLI をインストール（無い場合のみ）
```bash
$ npm i -g @nestjs/cli
```

パッケージインストール
```bash
$ npm install
```

起動
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

テスト
```bash
# 単体テスト
$ npm run test

# e2e テスト
$ npm run test:e2e

# カバレッジ表示
$ npm run test:cov
```

## その他

```bash
# ヘルプ
$ nest --help

# 生成コマンドヘルプ
$ nest generate --help
```
